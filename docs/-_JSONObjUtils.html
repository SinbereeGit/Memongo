_JSONObjUtils

_JSONObjUtils

Utility class for manipulating plain JSON-like objects using dot-notation paths. Provides helper methods for: - Retrieving nested values - Setting nested values - Performing partial updates - Cloning JSON objects All path-based operations support **dot notation** (e.g. `"user.profile.name"`).

Constructor

new _JSONObjUtils()

Source:
  • memongo.js, line 70
Example
const obj = {
  user: {
    profile: {
      age: 20
    }
  }
};

JSONObjUtils.getByPath(obj, "user.profile.age"); // 20

JSONObjUtils.setByPath(obj, "user.profile.age", 21);

JSONObjUtils.update(obj, {
  "user.profile.age": (age) => age + 1
});
